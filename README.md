# Tribes

A simple REST API built with Go that provides information about Kenyan tribes, their regions, cultural tags, population data and more.<br>
Available Endpoints:

    		GET all tribes
    		GET /tribes

    		GET tribe by ID
    		GET /tribes/1

    		CREATE tribe
    		POST /tribes

    		UPDATE tribe
    		PUT /tribes/1

    		DELETE tribe
    		DELETE /tribes/1
